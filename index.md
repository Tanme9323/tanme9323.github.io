**Does the app run when I turn on my computer?**
Yes, the software starts with Windows by default unless you change this in the settings menu.

**Can I monitor multiple accounts?**
The app allows you to link one account per instance. If you manage multiple accounts, open the settings to switch keys.

**How do I update the application?**
The app checks for updates automatically. You receive a notification when a new version is ready for download.

**Is my data private?**
The application encrypts your API keys on your local machine. No data leaves your computer except for the necessary status checks sent to your servers.

**What do I do if the app shows a connection error?**
Check your internet connection first. Then, verify that your API key is correct in the settings menu. If problems persist, restart the application and check the server status page on the main xCloud website.

**How much data does the app use?**
The monitoring service uses very little bandwidth. It sends small packets to verify server health and does not stream media or large files.
